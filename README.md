# AgriFusion AI Lite

A React application for agricultural marketplace and AI-powered services.

## Project Structure

```
agrifusion-ai-lite/
├── src/
│   ├── api/
│   │   └── supabase.js          # Supabase client
│   ├── components/
│   │   └── ui/                   # shadcn components
│   │       ├── button.jsx
│   │       ├── card.jsx
│   │       ├── input.jsx
│   │       ├── select.jsx
│   │       ├── badge.jsx
│   │       ├── tabs.jsx
│   │       └── ... (copy from base44 or install shadcn)
│   ├── pages/
│   │   ├── Home.js
│   │   ├── QualityChecking.js
│   │   ├── MarketAdvisor.js
│   │   ├── Marketplace.js
│   │   ├── AddListing.js
│   │   ├── AgriBot.js
│   │   └── Profile.js
│   ├── lib/
│   │   └── utils.js              # Utility functions
│   ├── Layout.js
│   ├── App.js
│   ├── index.js
│   └── index.css
├── package.json
└── README.md
```

## Setup

1. Install dependencies: `npm install`
2. Start the development server: `npm start`

## Features

- Quality Checking
- Market Advisor
- Marketplace
- Add Listing
- AgriBot
- Profile Management
